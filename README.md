# Sovera Homebrew tap

[![version](https://img.shields.io/github/v/release/sovera-dev/sovera-toolkit?label=pulse&color=brightgreen)](https://github.com/sovera-dev/sovera-toolkit/releases/latest)
![release](https://img.shields.io/badge/release-published-brightgreen)
[![license](https://img.shields.io/badge/license-MIT-brightgreen)](LICENSE)

Homebrew tap for [Sovera](https://github.com/sovera-dev) command-line tools
(macOS and Linuxbrew).

## Usage

```sh
brew install sovera-dev/tap/pulse
```

Or tap first, then install:

```sh
brew tap sovera-dev/tap
brew install pulse
```

The casks in this repository are generated automatically on every release by
[GoReleaser](https://goreleaser.com) from
[sovera-dev/sovera-toolkit](https://github.com/sovera-dev/sovera-toolkit).
